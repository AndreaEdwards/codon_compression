Welcome to the LASER Database

The LASER (Learning Assisted Strain EngineeRing) database is is a comprehensive internet resource for information on strategies in metabolic strain engineering.

Our goal is to catalog all known metabolic engineering strategies currently used to develop microbial strains for the production of valuable metabolites. The database currently contains data from over 200 papers using Escherichia coli as a chassis organism, and studies using Saccharomyces cerevisiae are rapidly being incorporated as well. Each database record contains information concerning the design, implementation, and result (product titer, yield) obtained from a given strain design. In the future, this curated database will be used to implement algorithmic approaches to strain engineering that combine human intuition, metabolic modeling, and random approaches to propose metabolic engineering interventions.
