Welcome to the Codon Compression web tool

Saturation mutagenesis is employed in protein engineering and genome-editing effots to generate libraries that span amino acid design space. Our codon compression web tool allows users to compress degenerate codons in a user specified manner. The output provided will be a unique collection of codons for any desired amino acids.
